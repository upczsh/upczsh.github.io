Spring 缺点

  • 配置过于繁琐（每次集成别的如Mybatis 都需要相关的配置）

SpringBoot

  • 简化初始搭建以及开发过程
  • 不再需要定义样板化的配置（如配置数据库）
  • 快速应用开发领域

Spring SpringMVC SpringBoot

  • Spring 最初利用IOC 和 AOP 解耦合（最核心）
  • 按照这种模式搞了 MVC 框架
  • 写样板代码麻烦 就搞了 SpringBoot
  • SpringCloud 是在SpringBoot 基础上诞生的

SpringBoot 核心特点

  • 开箱即用（如main函数一般）
  • 约定优于配置（如端口号8080）

SpringBoot 版本介绍

  • CURRENT
  • GA 代表最新的发布稳定版本（不会更改）
  • SNAPSHOT 通常被修改(会更改代码)

Spring2.0

  • 环境升级 要求Java版本最低为JDK8 Tomcat 8+
  • 默认软件的优化
  • 新的功能（HTTP2.0的支持）

实现小案例

项目结构解析

  • SpringBoot的基础结构共三个文件：入口(SpringBootApplication)，配置文件（resources下），测试入口（test）
  • 生成的Application和ApplicationTests 类都可以直接运行来启动当前创建的项目
  • pom.xml SpringBoot 的版本和项目的一致：2.x.x.RELEASE(RELEASE代表发行版本)

Controller各种传参

  • Controller 演示传参
    
package com.example.demo;


import org.springframework.web.bind.annotation.*;

/**
 *  演示各种传参形式
 */
@RestController  //在普通的Controller之上有Rest能力
@RequestMapping("/prefix") // 每个接口都拥有prefix前缀
public class ParaController {
    @GetMapping("/firstrequest") // url get请求
    public String firstRequest(){
        return "第一个Spring Boot 接口";
    }

    @GetMapping({"/requestpara"})
    public String requestpara(@RequestParam Integer num)
    {
        return "para from request: "+num;
    }

    @GetMapping({"/para/{num}"})
    public  String pathpara(@PathVariable  Integer num) //从URL中寻找并且绑定 传参不必num=..
    {
        return "para from path: " +num ;
    }

    //多URL 用法

    @GetMapping({"/multiurl1","/multiurl2"})
    public  String multiurl(@RequestParam  Integer num) //从URL中寻找并且绑定 传参不必num=..
    {
        return "para from path: " +num ;
    }

    /**
     * 不传参默认值为0
     * @param num
     * @return
     */
    @GetMapping({"/required"})
    public String required(@RequestParam(required = false,defaultValue = "0") Integer num)
    {
        return "para from request: "+num;
    }
}

Web项目的三层结构

  • Controller 职责
    只做一些简单的逻辑判断
  • Service 职责
    做主要业务 保持Controller的简洁 可被Controller重复调用
  • DAO层职责
    放和数据相关（CRUD）

Service和DAO的编写

学生信息查询案例

  • Controller

    	package com.example.demo;

import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RequestParam;
import org.springframework.web.bind.annotation.RestController;

import javax.annotation.Resource;

/**
 *   学生Controller
 */
@RestController
public class StudentController {
    @Resource
    StudentService studentService;
    @GetMapping({"/findstudent"})
    public  String student(@RequestParam Integer num){
        Student student = studentService.findStudent(num);
        return student.toString();
    }
}

  • Service

    	package com.example.demo;

import org.springframework.stereotype.Service;

import javax.annotation.Resource;

/**
 * 学生Service
 */
@Service
public class StudentService {
    @Resource
    StudentMapper studentMapper ;

    public  Student findStudent(Integer id){
        Student byId = studentMapper.findById(id);
        return byId;
    }
}
  • DAO
    import org.apache.ibatis.annotations.Mapper;
import org.apache.ibatis.annotations.Select;
import org.springframework.stereotype.Repository;

/**
 * 学生Mapper
 */
@Mapper
@Repository // 识别为Mapper对象
public interface StudentMapper {
    @Select("select *from student where id = #{id}") // #{}来对应
    Student findById(Integer id);
}
